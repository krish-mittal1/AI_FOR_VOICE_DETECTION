# AI Fraud Detection

Install [Python 3.13+](https://www.python.org/downloads/) and [uv](https://docs.astral.sh/uv/)

Install dependencies:

```bash
uv sync
```

Run the API:

- For development

```bash
uv run fastapi dev app/main.py
```

- For production

```bash
uv run fastapi run app/main.py
```
